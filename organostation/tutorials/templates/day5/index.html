Data analysis

After recording electrical signals, minimal quantification and visual signals ensure the proper functioning of the equipment and biological preparation. This tutorial will guide you to visualize and briefly quantify relevant signals before defining a specific software pipeline for evaluating a scientific hypothesis.

We will use Python (recommended version > 3.8) because it provides an easy programming framework available on all platforms.

1. Software requirements

Your computer already has a built-in python system. We don't want to interfere with it. That's why we will create a dedicated environment where we will make our analyses and visualizations. To have dedicated python environments, we need to download and install an environment called Anaconda.

1.1 Definitions

Anaconda is a package manager for Python that permits the easy handling of python libraries in dedicated environments.
